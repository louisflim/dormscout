# 🏠 DormScout – React Booking App

DormScout is a React-based booking platform that allows users to browse, book, and manage dorm listings بسهولة. It includes authentication, messaging, notifications, and booking features.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🏘️ Browse and view listings
- 📅 Book dorms
- 💬 Messaging system
- 🔔 Notifications
- 👤 User profiles
- ⭐ Reviews & ratings
- ⚙️ Settings & support

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **State Management:** Context API
- **Routing:** React Router
- **HTTP Requests:** Axios / Fetch API
- **Styling:** CSS / Tailwind (update if needed)

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/dormscout.git
```


2. Navigate to the project:
```bash
cd dormscout/frontend
```

3. Install dependencies:
```bash
npm install
```


4. Start the development server:
```bash
npm start
```
